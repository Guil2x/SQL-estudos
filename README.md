# 🗄️ SQL Estudos

Repositório dedicado aos meus estudos e prática em SQL (SQLite).

## 🎯 Objetivo

Este repositório documenta minha jornada de aprendizado em SQL, com foco em:
- Consultas com SELECT, WHERE, ORDER BY
- Junções (INNER JOIN, LEFT JOIN)
- Agregação (GROUP BY, HAVING, COUNT, SUM)
- Subconsultas
- Manipulação de dados (INSERT, UPDATE, DELETE)

## 📂 Estrutura

- exercism/ - Exercícios resolvidos do Exercism.org (SQLite track)
- consultas/ - Queries que eu criar por conta própria
- projetos/ - Projetos pequenos com dados reais (em breve)

## 🚀 Como executar

- Clone o repositório, entre na pasta e execute:
- git clone https://github.com/Guil2x/sql-estudos.git
- cd sql-estudos
- sqlite3 nome_do_banco.db
